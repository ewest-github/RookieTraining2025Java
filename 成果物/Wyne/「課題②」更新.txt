JSP　と　サーブレットは、Webアプリケーション開発に用いられるJavaのプログラムです。

JSPはHTMLにJavaのコードを埋め込み、サーブレットはWebブラウザからのリクエストに応じた処理を行います。

流れとしてはクライアントがJSPを要求すると、WebサーバはJSPをサーブレットにコンパイルします。 後は「サーブレットの動作概要」と同様に、サーバ側の処理が行われます。

 Webアプリケーションで使用できる言語はJava以外にもありますが、ほかにどのような言語が用いられている。
フロントエンド（ユーザー側）
JavaScript
typescript

バックエンド（サーバー側）
python
php
ruby
C#
Node.js

データベース言語
SQL(MySQL,PostgreSQL,SQLite)
